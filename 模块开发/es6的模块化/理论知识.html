es6之前没有模块化，为了解决问题，提出了common.js/seajs/requirejs

es6汲取各种优点，支持异步加载

  1. 新增两个关键字:export和import
  2. export 暴露 export default 为模块指定默认输出，一个模块只能使用一次 import 引入 import {bar,foo,test,obj} from './util.js' foo()
  3. es6模块运行机制 动态引用(import {bar,foo,test,obj} from './util.js'), 引用进来后,变量不会被缓存，而是成为一个执行被加载模块的引用， 脚本执行时，根据只读引用，到被加载的那个模块中去取值
